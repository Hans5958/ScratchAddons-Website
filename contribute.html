Contribute

Contributing to Scratch Addons is easy! Simply go to the GitHub repository for Scratch Addons and fork it! Once you have made changes to it that you would like in the official addon feel free to start a PR (Pull Request) with your changes. If World_Languages and apple502j review your code and think it's good your changes will be put into the official addon!

If you can't code or would just like to help with the discussion and suggestions for the addon feel free to comment and contribute in the issues tab of Scratch Addons. There you can suggest new addons to be added, help with the coding a little, and much more!
